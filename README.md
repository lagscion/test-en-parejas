# Descripción de el proyecto 

Se creó un programa de inventario para la venta de empanadas de  Pepa.
El programa solicita la opción que se desea realizar.
Luego se encarga de solicitad la información solicitada dependiendo de la opción.
La información se guarda como diccionario en archivo json.
En el caso de agregar o editar la información, el programa guardara y cambiara la información en el archivo json.
Si por el contrario es para eliminar una empanada, solicitara la empanada que se desea eliminar y la borrare de el archivo json.
Al finl el programa se repite, hasta que el usuario desea terminar con el proceso y salir de el programa.

## Instrucciones de instalación y ejecución
Se descarga toda la información.
Se inicia el programa desde el modulo main. 
Se puede ignorar el archivo json ya que el programa se encargara de crear uno nuevo como si no tuviera información.
